## Download
```
git clone https://github.com/Aaaaaaaty/vue-im
```
## Deploy
### Client
```
npm install
```
### Server
```
npm install -g nodemon //instead of 'node' to run your code
cd ./server/
npm install
```
### Database
```
brew install mongodb

```
## Run
### Client
```
npm run dev
```
### Server
```
nodemon ./server/bin/www
```
### Database
```
sudo mongod --dbpath=/data/db --port=27017
```
